# Программа для фильтрования строк по количеству символов

**Описание функционала:**
1. Программа при старте спрашивает пользователя хочет ли он ввести массив строк самостоятельно или использовать пример.
2. Создает новый массив строк и присваевает ему ссылку либо на введенный массив , либо на массив заданный в программе для примера.
3. Считает количество строк с длинной больше либо равной 3, записывает это значение в переменную и возвращает ее.
4. На основании переменной из п.3 создает новый массив строк.
5. Заполняет его циклом for
6. Выводит на печать.

## Карта программы:
Проект разделен на классы Решение (Solution) и Вспомогательные методы (Utility).
Solution содержит метод **Main**, где представлена основная логика и вызов методов и метод  
**MakeDecision**, отвечающий за принятие пользователем решения по вводу массива самостоятельно или использование примера.
## Utility содержит методы:
**ProcessedStringCount** - принимает массив строк и целое число, которое будет минимальным ограничением по длине строки. Возвращает переменную stringCount - количество строк, которые содержать символов больше, либо равное целому числу wordlenght, переданному в метод.  
**MakeArray** - принимает массив строк (тот же, что и предыдущий метод ProcessedStringCount, переменную stringCount из этого метода и тот же wordlenght, заданный изначально. Так же вначале объявляется переменная j, отвечающая за счетчик заполнения нового массива. Заполняются только те значения, которые больше, либо равны значению wordlength. Метод возвращает массив строк resultArray.  
**PrintArray** - принимает абсолютно те же значения, что и метод MakeArray. То есть массив строк, количество строк больше, либо равное wordlength, и сам wordlength. Массив подаваемый, через цикл for выводится в консоль, а остальные переменные используются для форматированного вывода текстового заголовка.
